# Number-Guessing



A simple and interactive Number Guessing Game built with Python. The objective is to guess the randomly generated number within the fewest attempts possible. This beginner-friendly project demonstrates Python fundamentals such as loops, conditionals, user input, and random number generation.

## ✨ Features
- Randomly generated number<br>
- Hints for higher or lower guesses<br>
- Unlimited guessing attempts<br>
- User-friendly console interface<br>
- Beginner-friendly Python project<br>

## 🛠️ Technologies Used
- Python 3
